# Bazar

Installing NPM dependencies:
```
npm install
```

Running the app:

```
meteor
```

## Testing

Running all tests using `chimp`:

```
npm run-script chimp
```

Running tagged tests  with `--watch ` option:
```
npm run-script chimp:watch
```

## Deployment

Deploying app to Galaxy:

```
npm run-script deploy
```
